# Repository Figures

This folder contains the main visual materials used in the GitHub README.

## Included Files

- `system-architecture.png` – main system architecture showing Customer Wallet, Retailer POS, and Local Security Server.
- `payment-authorization-flow.png` – detailed end-to-end authorization and storage flow.
- `payment-sequence-simple.png` – simplified transaction sequence diagram from the report.
- `security-log-pipeline.png` – event logging pipeline from Android to local database and admin console.
- `scenario-decision-accuracy.png` – estimated scenario accuracy chart used in the report.
- `successful-payment-timing.png` – timing chart for a successful payment flow.

These images are intended for documentation and presentation of the project structure, workflow, and test outcomes.
