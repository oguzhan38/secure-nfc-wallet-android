# Portfolio Description

## Short Project Title

```text
Secure NFC Wallet Android Application
```

## Short CV Description

```text
Developed an Android NFC wallet prototype using HCE, APDU communication, a local Flask security server, JWT authentication, nonce-based replay protection, tamper simulation, security logs and receipt history.
```

## Longer GitHub / LinkedIn Description

```text
Secure NFC Wallet is an academic network security project that demonstrates NFC-based mobile payment between two Android devices. The system includes a Customer Wallet, a Retailer POS and a local PC security server. It records APDU events, nonce values, timing metrics, authorization results, replay attempts, tamper alerts and receipt history for demonstration and analysis.
```

## Technologies

```text
Android Java, NFC, HCE, APDU, Flask, SQLite, JWT, AES, SHA-256, Gradle
```

## Main Security Concepts

```text
Authentication, authorization, encrypted payloads, nonce validation, replay protection, payload tamper detection, security logging, local monitoring console.
```
